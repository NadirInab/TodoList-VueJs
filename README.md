# Vue 3 : To Do List

### Clone the project in your local machine :
```bash
    git clone https://github.com/NadirInab/TodoList-VueJs.git
```

### To install the dependencies run : 
```bash
    cd Project-Name :
    npm install 
```

### To run the project run : 
```bash
    npm run dev  
```
